## ⚠️ 重要提醒：目录名与内容错位！

`md_split/` 是 reorganized 版切出来的，**目录名和里面文件的实际内容严重错位**。例如：
- `Transformer.md` 在 **`04_RAG/`** 里（不在 LLM 模型结构）
- `量化 / 蒸馏 / 剪枝 / 推理框架 / 二值化` 在 **`07_Agent/`** 里（不在推理加速）
- `Function_Calling / MCP / Agent 性能评估` 在 **`06_大模型岗位介绍/`** 里
- `SFT 和 RLHF 的区别` 在 **`07_Agent/`** 里；而 `10_模型微调/` 里有 SFT 方案、强化学习方案、SKILL

**所以：永远按下方的"知识点 → 文件"表定位，别按目录名猜。** 找不到时，退回读该目录的 `_FULL.md`（全量）再定位。

所有路径根目录：`<YOUR_LEARNING_MATERIALS_DIR>/md_split/`
